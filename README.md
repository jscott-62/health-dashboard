# Health Dashboard

Personal health tracking dashboard hosted on GitHub Pages.

## Features
- Daily habit checklists with streak tracking
- Weight, BMI, and body composition tracking
- Exercise and workout logging
- Water intake and supplement tracking
- Meal logging
- Goal progress bars and weekly comparisons
- Cross-device sync via GitHub API

## Setup
1. Visit the dashboard at https://jscott-62.github.io/health-dashboard/
2. Click the sync status indicator to connect a GitHub personal access token
3. Start tracking!

## Tech
Single-file HTML app. No frameworks, no build step, no external dependencies.
